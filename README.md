**AgePredictor** - Full pipeline of predicting biological age from any GEO illumina beadchip matrix file 

**BarnesHut** - Simulation of a gravity simulator using the barnes hut method to mimic galaxies colliding, thousands of stars interacting, and planets orbitting with their moon

**Boids** - Simulation of boids interacting in a set space built off collisions and grouping 

**Sandpiles** - Simulation of a discrete diffusion simulation using a toppling method with both parallel and serial implementations 

**GrayScott** - Simulation of a reaction - diffusion simulation using the gray scott method
